# Ficha-Cadastro
É um projeto desenvolvido para permitir o cadastro e a atualização de fichas para jogar RPG. Partindo da premissa que códigos tem que resolver problemas, escolhi esse projeto para resolver um problema meu e do meu grupo que é montagem de fichas para jogarmos. Sempre que vamos montar qualquer ficha, precisamos fazer as contas dos pontos, ver qual rank dá qual habilidade e fazer tudo manualmente. A ideia do projeto é permitir que isso seja feito com mais facilidade. 

## Naruto RPG
O projeto é voltado para um RPG de Naruto onde divimos todo o sistema em evoluções que passam pelos ranks, com distruibão progressiva de atributo, de perícias e ganho de maestrias. A obteção de jutsus (habilidades/magias do universo de Naruto) também será implementado. É um sistema que foi desenvolvido e maturado por mais de uma década, utilizando desde os fóruns do Orkut e, após seu fim, migrando para o Yoble e atualmente comportando tudo no Docs.
